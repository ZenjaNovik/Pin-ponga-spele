# Pin-ponga-spele
Šī programma ir Pong spēle, kas izveidota ar Python un pygame bibliotēku, kur divi spēlētāji kontrolē paddle. Spēlētāji izmanto taustiņus – kreisais paddle ar W un S taustiņiem, labais ar augšup un lejup bultiņām –, un bumba atsitās pret paddle vai sienām, lai turpinātu spēli. Spēle darbojas nepārtraukti, līdz lietotājs aizver logu.
